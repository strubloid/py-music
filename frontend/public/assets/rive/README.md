# Rive runtime source

`city-transit.riv` is the Rive-maintained vehicle demonstration file downloaded from `https://cdn.rive.app/animations/vehicles.riv` for the Living Music City transit layer.

The surrounding district composition, motion policy, fallback artwork, styling, interaction, and sound design are project-authored. The third-party source asset remains isolated here so it can be replaced with a project-authored Rive export without changing runtime code.
